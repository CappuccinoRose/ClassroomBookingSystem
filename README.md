# Classroom Booking System

教室预订系统 - 基于 Flask + Vue 3 的教室资源管理平台。

## 技术栈

- **后端**: Python + Flask + SQLAlchemy + JWT
- **前端**: Vue 3 + Element Plus + Vite
- **数据库**: SQLite / MySQL

## 项目结构

```
classroom-booking-backend/    # Flask 后端
classroom-booking-frontend/   # Vue 3 前端
```

## 快速开始

### 后端启动

```bash
cd classroom-booking-backend
pip install -r requirements.txt
python run.py
```

### 前端启动

```bash
cd classroom-booking-frontend
npm install
npm run dev
```
